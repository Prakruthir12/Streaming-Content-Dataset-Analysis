# 📺 Streaming Content Dataset Analysis

A **data analysis project** that explores streaming platform content such as **Movies and TV Shows** using a dataset containing metadata like title, director, cast, release year, rating, and genre.

This project focuses on **exploratory data analysis (EDA)** to understand patterns and trends in streaming platform content.

---

## 📊 Dataset Overview

The dataset contains information about **movies and TV shows available on streaming platforms**.

### Key Columns

- **show_id** – Unique identifier for each show  
- **type** – Content type (Movie or TV Show)  
- **title** – Name of the movie or TV show  
- **director** – Director of the content  
- **cast** – Main actors/actresses  
- **country** – Country where the content was produced  
- **date_added** – Date when the title was added to the platform  
- **release_year** – Year the title was released  
- **rating** – Age/content rating  
- **duration** – Movie duration or number of seasons  
- **listed_in** – Genre/category  
- **description** – Short summary of the content

## 🎯 Project Objectives

The main goals of this project are:

- Clean and preprocess the dataset
- Analyze the **distribution of Movies vs TV Shows**
- Identify **top countries producing content**
- Explore **content trends across years**
- Examine **popular genres and ratings**
- Analyze **directors and cast appearances**
